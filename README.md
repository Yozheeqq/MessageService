# MessageService
Данный проект разработан на ASP.NET WEB API\n
Проект позволяет создать примитивный пример для работы с отправкой и получением сообщений.\n
Для хранения списка пользователей и сообщений используются json файлы с дальнейшей их (де)сериализацией.\n
Для упрощенного понимания работы с сервисом использованы xml комментарии встроенные в swagger.
